# Installation

## Stable release

To install orderflow-alpha, run this command in your terminal:

```sh
uv add orderflow-alpha
```

Or if you prefer to use `pip`:

```sh
pip install orderflow-alpha
```

## From source

The source files for orderflow-alpha can be downloaded from the [Github repo](https://github.com/JS195/orderflow-alpha).

You can either clone the public repository:

```sh
git clone https://github.com/JS195/orderflow-alpha
```

Or download the [tarball](https://github.com/JS195/orderflow-alpha/tarball/main):

```sh
curl -OJL https://github.com/JS195/orderflow-alpha/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd orderflow-alpha
uv sync
```
